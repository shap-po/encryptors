# Encryptors

A collection of encryption algorithms implemented in JS with SvelteKit frontend.

## How to run

1. Clone the repository

    ```bash
    git clone https://github.com/shap-po/encryptors.git
    ```

1. Install dependencies

    ```bash
    cd encryptors
    npm install
    ```

1. Run the project

    ```bash
    npm run dev
    ```
